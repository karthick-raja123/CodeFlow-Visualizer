IF SOMETHING GOES WRONG:

1. Server Won't Start:
   - Check port 8001 is not in use: python restart_server.py
   - Verify Python version: python --version (3.10+)
   - Check main.py is present in backend folder

   - Restart server: python restart_server.py
   - Re-run tests: python qa_test_orchestrator.py

   - Run Phase 2 audit: python qa_phase2_security_audit.py

   - Review logs for NameError exceptions

4. Performance Issues:
   - Check response times: Monitor <155ms average
   - Check timeout events: <1% of requests
